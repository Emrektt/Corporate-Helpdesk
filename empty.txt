Proje adı: Corporate Helpdesk

Şirket çalışanlarının IT, insan kaynakları, finans, satın alma ve idari işler taleplerini oluşturabileceği; ilgili departmanların bu talepleri yönetebileceği kurumsal destek uygulaması.

Stajyer üç hafta sonunda şu süreçlerin tamamını görmüş olacak:

İhtiyaç analizi
→ Sistem tasarımı
→ Database tasarımı
→ Backend API
→ Frontend
→ Microsoft ile giriş
→ Yetkilendirme
→ Test
→ Docker
→ CI/CD
→ Deployment
→ Dokümantasyon

FastAPI; Python type hint’leriyle API geliştirmeye yönelik bir framework’tür ve otomatik OpenAPI/Swagger desteği sunar. React tarafında TypeScript kullanılabilir. MSAL ise Microsoft Identity Platform’dan token alınarak SPA ve API güvenliği sağlamak için kullanılacaktır.

3. Microsoft MSAL mimarisi

Sistemde uygulamanın kendi şifre ekranı olmayacak. Kullanıcı şirket Microsoft hesabıyla giriş yapacak.

Kullanıcı
   ↓
React uygulaması
   ↓
MSAL React
   ↓
Microsoft Entra ID
   ↓
Access Token
   ↓
FastAPI
   ↓
Token doğrulama
   ↓
PostgreSQL

MSAL, Microsoft Identity Platform’dan güvenlik token’ı alınmasını sağlar. Bu token Microsoft servislerine veya geliştirilen özel API’ye erişimde kullanılabilir.

React tarafında kullanılacak paketler
npm install @azure/msal-browser @azure/msal-react

Temel yapı:

MsalProvider
LoginButton
LogoutButton
ProtectedRoute
AcquireTokenSilent
Axios interceptor

React uygulaması API’ye istek atarken header’a token ekleyecek:

Authorization: Bearer ACCESS_TOKEN
FastAPI tarafında yapılacaklar

Backend gelen token üzerinde şunları kontrol edecek:

Token imzası geçerli mi?
Token süresi dolmuş mu?
Issuer doğru Microsoft tenant mı?
Audience bizim API’nin Client ID değeri mi?
Kullanıcının hesabı aktif mi?
Kullanıcının gerekli rolü var mı?
Entra ID üzerinde gereken kayıtlar
